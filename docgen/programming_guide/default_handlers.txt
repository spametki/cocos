Handlers por Defecto
====================

cocos2d viene con handlers por defecto para algunas acciones.

En Windows y Linux, los atajos son
CTRL + tecla, pero en Mac, el atajo es
COMMAND + tecla.

 - CTRL + f

    Cambia a pantalla completa

 - CTRL + s

    Hace una captura de pantalla de la ventana actual.

    Se guardará un archivo con nombre 'screenshot-xxxxx' en el
    directorio actual de ejecución.

 - CTRL + p

    Pausa / Continuar

 - CTRL + w

    Activa / Desactiva el modo wire-frame

 - CTRL + x

    Activa / Desactiva la información de cuadros por segundo

 - CTRL + i

    Activa / Desactiva el intérprete de python embebido
