Introducción
============

cocos2d es un framework para crear juegos 2d, demos y otras aplicaciones
gráficas/interactivas.

cocos2d simplifica el desarrollo de un juego en estas areas:

    - **Control de flujo:** Maneja el control de flujo entre distintas escenas de forma fácil
    - **Sprites:** Sprites rápidos y fáciles
    - **Acciones:** Basta con decirle al sprite lo que quieres que haga. Composición de acciones como mover, rotar, escalar, etc
    - **Efectos:** Efectos como ondas, retorcer, lente y mucho más
    - **Sistemas de partículas:** Explosiones, fuegos artificiales, meteoros, humo y más
    - **Mapas en mosaico:** Soporte para mapas en mosaico rectangulares y hexagonales
    - **Detección de colisiones:** Soporte de colisiones básico en python puro
    - **Transiciones:** Cambia de escena a escena con estilo
    - **Menús:** Clases incorporadas para creación de menús
    - **Render de texto**
    - **Documentación:** Guía del Programador + Referencia de la API + Numerosos tests sencillos que muestran como usarlo
    - **Intérprete de python incorporado:** Para depuración
    - **BSD license:** Úsalo sin preocuparte
    - **pyglet based:** Sin dependencias externas
    - **OpenGL based:** Acelerado por hardware
