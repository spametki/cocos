El Intérprete de Python Embebido
================================

Se puede activar un intérprete de python presionando
las teclas CTRL + i en Windows y Linux y con
COMMAND + i en OS/X.

Los ``locals()`` del intérprete de python son:

    - ``cocos``
    - ``director``
    - ``self``

Además el intérprete tiene algunas propiedades
que se pueden cambiar en ejecución, por ejemplo:

    - ``opacity``
    - ``color``
    - ``cfg['code.font_size']``
    - ``cfg['code.font_name']``
    - ``cfg['code.font_color']``

Por ejemplo::

    # 0 significa sin opacidad
    # 255 significa totalmente opaco
    self.opacity = 50

    # modificando el color de la sección a uno rojizo
    self.color = (204, 32, 32 )

    # cambiando el color de la fuente a blanco
    self.cfg['code.font_color'] = (255,255,255)

    # cambiando el tamaño de la fuente
    self.cfg['code.font_size'] = 24

    # cambiando el nombre de la fuente
    self.cfg['code.font_name'] = 'Courier New'

    # obteniendo la escena actual:
    current_scene = director.scene

Una vez que se activa el intérprete, este permanecerá en memoria.
De aquí en más, puedes ocultarlo o mostrarlo presionando las teclas
(de nuevo) CTRL+i (o COMMAND+i).
